idiot help: print help for a command

Usage: idiot help <command>

Arguments:
   <command>   the command to print help for

Commands:
   branch [-d <branch>]
   cat-file {-p|-t} <address>
   commit <tree> -m "message" [(-p parent)...]
   commit-tree <tree> -m "message" [(-p parent)...]
   hash-object [-w] <file>
   help
   init
   rev-parse <ref>
   switch [-c] <branch>
   write-wtree
